# Kayijuka Olivier Personal Website

This repository contains the personal website of Kayijuka Olivier, a Web Developer and IoT/Embedded Systems enthusiast with experience in ethical hacking and computer engineering.

## Project Overview

The website is designed to showcase Kayijuka Olivier's skills, projects, and contact information. It is fully responsive, mobile-friendly, and includes an Instagram booking option.

### Features

* **Home Page:** Introduction and personal statement.
* **Skills Page:** Lists technical skills and projects (with optional project images).
* **Contact Page:** Email, WhatsApp, GitHub links, and Instagram booking button.
* **Mobile-Ready:** Optimized for Redmi and other Android devices.
* **Custom Footer:** Personalized with "Kayijuka Olivier | Personal Website".
* **Optional PWA Support:** Manifest and Service Worker for home screen installation.

## Project Structure

```
/ (root)
├── index.html
├── skills.html  (optional, can be single-page scroll)
├── contact.html (optional, can be single-page scroll)
├── style.css
├── manifest.json  (for mobile install)
├── sw.js          (optional service worker for PWA)
└── images/       (all images including icons)
```

## How to Use

1. **Clone the repository:**

   ```
   git clone https://github.com/kayijuka-o/kayijuka-o.github.io.git
   ```
2. **Open `index.html` in a browser** to view the website.
3. **Upload to GitHub Pages** to go live.
4. **Optional:** Add PWA support for mobile devices (manifest + icons + service worker).
